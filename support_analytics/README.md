# Support Analytics

Esta carpeta almacena el trabajo del **Analista de Soportes**.

## Estructura

- `support_analytics/snapshots/`
  - Fotografías periódicas del estado de soporte.
  - Cada archivo resume temas principales, causas raíz y métricas cualitativas.
- `support_analytics/roadmap/`
  - Backlog de mejoras propuestas a partir de patrones detectados en soporte.

## Convenciones

- Idioma: español.
- Formato sugerido de snapshot:
  - Resumen ejecutivo.
  - Temas principales (1..N).
  - Riesgos / impacto.
  - Recomendaciones.
- Formato sugerido de roadmap:
  - Lista de ítems con: categoría, módulo, descripción, impacto, esfuerzo estimado (si se conoce), prioridad y dueño sugerido.
